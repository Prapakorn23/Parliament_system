# เริ่มใช้งานด่วน
## Quick Start Guide

---

## ⚡ เริ่มใช้งานภายใน 3 นาที

### **ขั้นตอนที่ 1: ติดตั้ง (1 นาที)**

#### **Windows:**
```bash
# Double-click ไฟล์นี้
scripts\install_windows.bat
```

#### **Linux/Mac:**
```bash
chmod +x scripts/install_linux_mac.sh
./scripts/install_linux_mac.sh
```

#### **หรือติดตั้งด้วยมือ:**
```bash
pip install -r requirements.txt
```

---

### **ขั้นตอนที่ 2: รัน (30 วินาที)**

#### **Windows:**
```bash
# Double-click ไฟล์นี้
run.bat
```

#### **Linux/Mac:**
```bash
chmod +x run.sh
./run.sh
```

#### **หรือรันด้วย Python:**
```bash
python app.py
```

---

### **ขั้นตอนที่ 3: ใช้งาน (1 นาที)**

1. **เปิดเบราว์เซอร์:**
   ```
   http://localhost:5000
   ```

2. **อัปโหลดไฟล์หรือพิมพ์ข้อความ:**
   - คลิก "เลือกไฟล์" → เลือก PDF หรือ TXT
   - หรือพิมพ์ข้อความในกล่อง

3. **กด "ตรวจสอบคำซ้ำ"**

4. **ดูผลลัพธ์!** 🎉

---

## 📊 ตัวอย่างทดสอบ

### **ทดสอบด้วยข้อความ:**

พิมพ์ข้อความนี้:
```
วันนี้มีการประชุมสภาเพื่อพิจารณางบประมาณด้านการศึกษา 
โดยมีการหารือเรื่องการจัดสรรงบประมาณสำหรับโรงเรียน 
การพัฒนาคุณภาพครู และการจัดหาทุนการศึกษาให้กับนักเรียน
```

**ผลลัพธ์ที่จะได้:**
- คำทั้งหมด: ~35 คำ
- คำที่ไม่ซ้ำ: ~28 คำ
- หมวดหมู่พบ: การศึกษา, เศรษฐกิจ, การเมือง

---

## 🎯 ทุกอย่างที่ต้องรู้

### **รองรับไฟล์:**
- ✅ `.txt` - ไฟล์ข้อความ
- ✅ `.pdf` - PDF ทุกประเภท

### **ฟีเจอร์หลัก:**
- 🔍 ตรวจสอบคำซ้ำ
- 📂 จัดหมวดหมู่คำ (16 หมวด)
- 📊 กราฟแสดงผล
- 💾 ส่งออก CSV

### **ขนาดไฟล์:**
- สูงสุด: 200MB (สามารถปรับได้ผ่าน .env)
- แนะนำ: < 50MB สำหรับประสิทธิภาพที่ดี

---

## ⚙️ การตั้งค่า (Optional)

### **เปลี่ยน Port:**
แก้ไข `config/config.py`:
```python
PORT = 8080  # เปลี่ยนจาก 5000
```

### **เปลี่ยนขนาดไฟล์สูงสุด:**
แก้ไขไฟล์ `.env`:
```env
MAX_FILE_SIZE_MB=500  # 500MB
```

หรือแก้ไข `config/config.py`:
```python
MAX_FILE_SIZE_MB = 500  # 500MB
```

## 🆘 ถ้ามีปัญหา

### **Problem 1: "ModuleNotFoundError"**
```bash
# ติดตั้ง dependencies ใหม่
pip install -r requirements.txt
```

### **Problem 2: "PDF ไม่สามารถแปลงได้"**
```bash
# ติดตั้ง PDF libraries
pip install PyPDF2 pdfplumber
```

### **Problem 3: "Port already in use"**
- ปิดโปรแกรมที่ใช้ port 5000
- หรือเปลี่ยน port ใน `config/config.py`
