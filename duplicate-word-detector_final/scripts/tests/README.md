# Test Scripts

โฟลเดอร์นี้เก็บ scripts สำหรับทดสอบระบบ

## ไฟล์ในโฟลเดอร์นี้

- **`test_db_connection.py`** - ทดสอบการเชื่อมต่อ database และการตั้งค่า
- **`test_date_extractor.py`** - ทดสอบการดึงวันที่จากข้อความ

## การใช้งาน

```bash
# ทดสอบการเชื่อมต่อ database
python scripts/tests/test_db_connection.py

# ทดสอบ date extractor
python scripts/tests/test_date_extractor.py
```
