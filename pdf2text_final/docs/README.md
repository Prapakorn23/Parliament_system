# 📚 Documentation

โฟลเดอร์นี้รวมเอกสารทั้งหมดของโปรเจค PDF to Text Converter & Summarizer

## 📂 โครงสร้างโฟลเดอร์

### `/api` - API Documentation
เอกสาร API และ Schema ของระบบ
- `API_DOCUMENTATION.md` - เอกสาร REST API ทั้งหมด
- `CHATBOT_API_EXAMPLES.md` - ตัวอย่างการใช้งาน Chatbot API
- `DATABASE_SCHEMA.md` - โครงสร้างฐานข้อมูล

### `/setup` - คู่มือติดตั้ง
คู่มือการติดตั้งและตั้งค่าระบบ
- `QUICK_SETUP_CHAT.md` - ติดตั้งระบบ Chat
- `QUICK_SETUP_POPPLER.md` - ติดตั้ง Poppler สำหรับแปลง PDF
- `QUICK_SETUP_TESSERACT.md` - ติดตั้ง Tesseract OCR

### `/guides` - คู่มือการใช้งาน
คู่มือการใช้งานฟีเจอร์ต่างๆ
- `CHAT_UI_GUIDE.md` - คู่มือใช้งาน Chat Interface
- `CHATBOT_GUIDE.md` - คู่มือระบบ Chatbot
- `CHATBOT_QUICKSTART.md` - เริ่มต้นใช้งาน Chatbot
- `DATABASE_MIGRATION.md` - คู่มือ Migration ฐานข้อมูล
